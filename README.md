# dotfiles

## 構成
```sh
├── .config
│   ├── mise
│   │   └── config.toml
│   ├── nvim
│   │   ├── init.lua
│   │   └── lua
│   │       ├── autocmds.lua
│   │       ├── keymaps.lua
│   │       ├── lsp_config.lua
│   │       ├── options.lua
│   │       └── plugins.lua
│   └── wezterm
│       └── wezterm.lua
├── .zsh_alias
├── .zshenv
├── .zshrc
├── init.sh
└── set_proxy.sh
```

## 実行
```sh
sh init.sh
sh install.sh
```
